Project · 项目愿景

我们要做什么

简单来说：让一群无人机自己组网、自己配合，完成协同任务。这是一个把“科幻片里的画面”变成现实的过程。

AI 让无人机更聪明

用简单的机器学习方法，让无人机能“看得见”周围环境，做出聪明的决策。不需要你是 AI 大神，我们会手把手带你入门。

无人机之间自己“说话”

不需要地面指挥，无人机之间通过自组网自动通信、交换信息。就像群鸟飞行，不用喊口令也能保持队形。

电脑里先“飞”一遍

先在仿真软件中模拟飞行和测试，验证通过后再上真机。既安全又省钱，试错成本几乎为零。

大家一起协作

不同方向的同学各司其职、互相配合。你不需要什么都会，只要在你负责的部分认真做就好。

Recruitment · 招募岗位

开放的岗位

01

算法 / AI 方向

训练无人机做简单的决策，比如避障、跟随。会用 Python 就行。

PythonPyTorch（加分）有兴趣就能来
02

通信 / 网络方向

让无人机之间能互相通信。了解网络基础概念就很棒了。

计算机网络基础C / C++（加分）在学也没关系
03

仿真 / 测试方向

搭建仿真环境、跑测试。这其实是个很好的入门切入点！

PythonLinux 基本操作零基础友好
04

嵌入式 / 飞控方向

跟硬件打交道，把代码烧到无人机上。喜欢动手的同学看这里。

C / C++单片机经验（加分）会焊接更好
05

感知 / 图像方向

帮无人机“看”东西，识别障碍物、追踪目标。

OpenCVPython学过图像处理即可
06

前端 / 后端开发

做一个控制面板或可视化界面，能看懂数据就行。

HTML / CSS / JSReact / Vue（加分）做过网页就行
07

硬件 / 结构方向

设计无人机的外壳、选传感器、画电路板。动手达人首选。

SolidWorksPCB 基础（加分）喜欢动手就行
08

项目管理 / 文档方向

整理资料、写文档、协调进度。细心靠谱最重要。

Office / LaTeX数据整理认真负责就行

Process · 加入流程

加入之后会怎样

不用担心跟不上，我们有完整的培养节奏。每一步都有学长学姐带你走，不会让你一个人摸索。

Step 01

聊聊看

先加微信 / 发邮件简单聊聊你的兴趣和方向，不设任何门槛。我们更看重热情和靠谱。

Step 02

入门学习

我们会提供学习资料和入门任务，学长学姐手把手带着你上手，从最基础开始。

Step 03

融入团队

参加定期组会，认识队友，明确你在项目中的角色和任务。节奏不会太赶。

Step 04

实战冲刺

一起打磨作品、准备答辩材料，冲刺挑战杯。这段经历会让你收获满满。
